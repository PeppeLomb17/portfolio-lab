# Contributing

## Branching
- main is protected
- create branches:
  - feat/<topic>
  - fix/<topic>
  - docs/<topic>
  - chore/<topic>

## Commits (Conventional Commits)
- feat: ...
- fix: ...
- docs: ...
- chore: ...

## Pull Requests
- small and focused
- must include: what, why, how tested
- no secrets or personal broker exports in the repo
